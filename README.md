# Базовый проект C++ и Allegro5

Данный репозиторий состоит из 3 частей.
1) `INSTALL.md` для установки `Allegro5` на `windows`/`linux` и в `VScode`/`Codeblocks`.
2) `ALLEGRO.md` краткий мини гайд по работе с библиотекой.
3) И сама репа уже готовый проект для работы с `Allegro5` на любой из выше перечисленных операционных систем но только в `VScode`. Для `Codeblocks` надо установить библиотеку самостоятельно а далее можно скачать содержимое `src` директории.


## Как работает проект

Он идет сразу с установленой библиотекой `Allegro5`, тремя примерами и вспомогательными файлами, которые далее будут нужны в работе.
Вспомогательных файлов 4, а именно:
- `AllegroUtil.cpp` и `AllegroUtil.hpp` для первых двух примеров и для первых занятий (до ООП);
- `engine.cpp` и `engine.hpp` для третьего примера и для последующей собственной реализации игрового движка (если таким его можно назвать).


### Компиляция на Windows

Нам нужен файл `build.bat`. Все `.cpp` файлы, которые используются в проекте должны быть указаны на месте `src/*.cpp` файлов внутри батника. Также можно изменить название испольняемого файла `build/windows/my_name.exe`.
```bat
g++ src/example_engine_3.cpp src/engine.cpp ^
-o build/windows/program ^
-std=c++17 ^
-Iallegro/include ^
-Lallegro/lib ^
-lallegro ^
-lallegro_main ^
-lallegro_primitives ^
-lallegro_image ^
-lallegro_font ^
-lallegro_ttf
```
Если не работает, попробуйте убрать `^` так, чтобы вся команда была в одну строку.

Запускаем в `cmd` и готово. Важно, чтобы испольняемый файл был в той де директории что и `.dll` файлы.

### Компиляция на Linux

Нам нужен файл `build.bat`. Все `.cpp` файлы, которые используются в проекте должны быть указаны на месте `src/*.cpp` файлов внутри скрипта. Также можно изменить название испольняемого файла `build/linux/my_name.exe`.
```sh
g++ src/example_engine_3.cpp src/engine.cpp \
-o build/linux/program \
-std=c++17 \
$(pkg-config --cflags --libs allegro-5 allegro_primitives-5 allegro_image-5 allegro_font-5 allegro_ttf-5)
```
Если не работает, попробуйте убрать `\` так, чтобы вся команда была в одну строку.

Запускаем в `bash` и готово.
